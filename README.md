# Myanmar Public Holiday Calendar 🇲🇲

A responsive, interactive React component for displaying Myanmar's public holidays. Built with React, TypeScript, and Tailwind CSS, this calendar provides a clean UI for navigating through months and years, complete with interactive tooltips for overlapping holiday events.

## ✨ Features

- **Multi-Year Support:** Navigate seamlessly between different years (currently configured for 2021–2026).
- **Interactive Tooltips:** Hover over highlighted dates to view specific holiday names. Handles multiple overlapping holidays gracefully.
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop views using a CSS Grid layout.
- **Skeleton Loading:** Displays a skeleton UI while fetching holiday data to ensure a smooth user experience.
- **Modern UI:** Glassmorphism effects, smooth transitions, and a polished color palette.

## 🛠️ Tech Stack

- **Framework:** React
- **Language:** TypeScript
- **Styling:** Tailwind CSS
